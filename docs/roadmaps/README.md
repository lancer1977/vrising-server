# V Rising Server Roadmaps

This directory tracks future work, release gates, and documentation follow-up for the `vrising-server` repo.

## Ladder

- `V0`: bootable Dockerized server, support/readback lane, deployment, and smoke checks
- `V1`: canonical support home lives in `../vrising-support`

The shared seam for the ladder lives in `Api.GameServerInterop`, and the
support-home contract lives in `../vrising-support`.

## Current Roadmap

- [Portfolio Roadmap](portfolio-roadmap.md) - the active lightweight roadmap for docs cleanup, validation, and packaging assumptions.

## Guidance

- Keep roadmap items small and verifiable.
- Prefer docs-first updates when behavior or deployment guidance changes.
- Link roadmap items back to the feature docs they depend on.
